# Best Messenger

## Description

Messenger on vanilla JS/TS.

## Requirements

NodeJS

## Install

```sh
npm install
```

## Start

```sh
npm start
```

Open http://localhost:3000 and enjoy

## Build

```sh
npm run build
```
